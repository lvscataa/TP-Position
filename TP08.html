1. ¿Cuál es la diferencia principal entre position relative y position absolute?

La diferencia entre position relative y absolute es que position relative permite desplazar un elemento desde su
posición original utilizando las propiedades top, bottom, left o right.
Mientras que position absolute permite colocar un elemento en una posición específica.


2. ¿Qué valor de position se utiliza por defecto para los elementos HTML?

Los valores de position que se utilizan para los elementos html son los valores top, bottom, right y left.


3. ¿Qué propiedad se utiliza para establecer la capa de apilamiento de elementos superpuestos?

La propiedad que se utiliza para establecer la capa de apilamiento es <Z-Index>.


4. ¿Cuándo se utiliza position fixed en CSS?

Se utiliza position fixed cuando queremos que un elemento se posicione en relación con la ventana del
navegador en lugar de su elemento padre.


5. ¿Cómo se comporta un elemento con position sticky al desplazarse?

Al desplazarse un elemento sticky lo que hace es que en un lugar determinado el elemento se puede mover hasta un punto limitado.
por ejemplo, la utilización de un position sticky en un header es que podés bajar por la página hasta un punto que el encabezado se queda estático en un lugar.


6. ¿Cuál es el valor de position adecuado para un elemento que se desea que siga el flujo normal del documento?

El valor de position adecuado es static.
